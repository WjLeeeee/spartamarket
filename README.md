> MypageActivity 의 대략적인 UI

- order_layout : 주문목록을 보여주는 레이아웃

  - 일단 틀을 작성하기때문에 ScrollView안에 widget들은 imageView로 넣어놨다.
  - 이후,더 복잡한 widget으로 대체할 가능성 있다.
